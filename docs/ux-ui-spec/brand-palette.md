# Brand Palette

**Confirmed by founder: 2 May 2026**

Flow AI's official brand colors — to be used across all UI surfaces including the chat widget and future CRM/dashboard interfaces.

---

## Color Tokens

| Token | Hex | Usage |
|---|---|---|
| `primary` | `#1B5E3F` | Deep forest green — CTAs, links, active states |
| `primaryDark` | `#144D33` | Darker hover state |
| `primaryLight` | `#E6F4ED` | Light tinted green — backgrounds, focus rings |

---

## Migration Note

**Previous default:** `#4F46E5` (indigo) is the old default color and is being replaced by `#1B5E3F` across all Flow AI UI surfaces.

All existing UI components should be updated to use the new green palette.

---

## Usage Guidance

### Primary (`#1B5E3F`)
Use for all interactive elements:
- Primary buttons
- Links and text links
- Widget button (chat launcher)
- Header backgrounds or accent elements
- Active navigation states
- Selected states

### Primary Dark (`#144D33`)
Use for hover and pressed states:
- Button hover backgrounds
- Link hover states
- Active/pressed button states

### Primary Light (`#E6F4ED`)
Use for subtle backgrounds and indicators:
- Focus rings on interactive elements
- Light background tints for emphasis areas
- Hover backgrounds for list items or cards
- Success message backgrounds (when paired with primary text)

---

## Scope

This palette applies to:
- **Chat widget** (all variants: embedded, floating, mobile)
- **CRM interface** (Phase 2)
- **Dashboard/analytics** (Phase 2)
- **Client-facing booking forms** (if applicable)
- **Flow AI platform marketing pages**

---

## Accessibility

All color combinations meet WCAG AA contrast requirements:
- `#1B5E3F` on white: 7.8:1 (AAA compliant for normal text)
- White on `#1B5E3F`: 7.8:1 (AAA compliant)
- `#144D33` on white: 10.5:1 (AAA compliant)
